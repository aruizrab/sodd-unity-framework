Class StateMachine

Manages the states in a state machine, handling state transitions and maintaining a state queue.

public class StateMachine : MonoBehaviour
Inheritance
object
StateMachine

Remarks

The StateMachine class is responsible for managing the lifecycle of states within a state machine. It provides mechanisms to transition between states, queue states for later execution, and maintain the current state.

Properties

CurrentState

Gets or sets the current state. Exits the previous state and enters the new state.

public State CurrentState { get; set; }

Property Value

State

The current state of the state machine.

Remarks

When setting the CurrentState, the state machine exits the previous state (if any) and enters the new state. If the new state is the same as the current state, no transition occurs.

ConsumeFirstFromQueue()

Sets the current state to the first state in the queue and removes it from the queue.

public void ConsumeFirstFromQueue()

Remarks

This method transitions the state machine to the first state in the queue and removes that state from the queue. If the queue is empty, no state transition occurs.

public void ConsumeLastFromQueue()

Remarks

This method transitions the state machine to the last state in the queue and removes that state from the queue. If the queue is empty, no state transition occurs.
