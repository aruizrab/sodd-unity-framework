Remarks

The PersistentScriptableObject class is designed to enable persistent storage and retrieval of Scriptable Variables within the SODD Framework. It serves as the base class for variables that need to be saved and loaded across game sessions, such as player settings and game states.

Each instance of a PersistentScriptableObject includes a toggle option in the Inspector, which indicates whether the specific variable should be persisted. This toggle option facilitates the management of variables in the Variable Repository, ensuring that only those marked for persistence are saved and loaded.
