Remarks

The InputActionIconProvider class is responsible for providing icons representing input actions based on the current control scheme. It listens for changes in the control scheme and updates the displayed icon accordingly. This class extends PassiveScriptableObject to ensure it is loaded into memory during runtime.

This class uses an InputIconRepository to fetch the appropriate icons for different input actions and control schemes. The icons are then applied to a target TMP_SpriteAsset for display.
